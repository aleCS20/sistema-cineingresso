
O Sistema foi desenvolvido em PHP em conjunto com a biblioteca CSS Bootstrap 

****   Sistema de controle de venda de ingressos de cinema on-line   ****

O sistema em questão é de grande relevância para o meio comercial em geral, no caso, 
ingressos de cinema online da empresa CineIngressos, um de sistema que permite ao cliente 
comprar ou consultar informações de filmes em exibição e posteriormente comprar ingressos, 
assim facilitará a vida do cliente em potencial, e evitaria alguns transtornos para o cliente e a 
empresa de cinema.
