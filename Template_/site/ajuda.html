Ajuda sobre o site

Aqui você encontra uma rápida consulta dos serviços do no nosso website.

Consultando Programação

Aqui você pode acompanhar toda a programação de filmes, sessões, datas horários do CineIngresso. Para acessá-lo você deve ir em Serviços, Programação, caso queira procurar por um filme em exibição basta digitar o nome do Filme no campo e depois clicar em Pesquisar.

Como se Cadastrar

Para se cadastrar como cliente clique no link Cadastre-se, preencha todos os campos obrigatórios, em seguida clique no botão Cadastrar você será redirecionado à página Home.

Caso queira desistir do cadastro clique no botão Cancelar.

Comprando Ingressos

Antes de tudo você deve realizar login no site, para comprar ingressos você deve acessar o link Serviços, Ingressos e Comprar, faça a escolha do filme, data, horário, tipo de ingresso e quantidade, feito isso clique no botão Comprar.

Caso queira desistir da compra clique no botão Cancelar.

Realizando Login

Você pode realizar login de várias formas, a mais simples é: Na página home clique em login em seguida você será redirecionado a tela de compra de ingresso para efetuar uma compra. Para sair clique em Sair.
